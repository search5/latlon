# Installation

## Requirements

- Python 3.6+
- [pyproj](https://pyproj4.github.io/pyproj/)

## Install via pip

```bash
pip install latlon3
```

## Install from source

```bash
git clone https://github.com/search5/latlon.git
cd latlon
pip install .
```

## Import

```python
import latlon

# or import specific classes
from latlon import LatLon, Latitude, Longitude, GeoVector
from latlon import string2latlon, string2geocoord
```
